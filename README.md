# Nicoke.Barber — Barbería & Style

Sistema de reservas online para barbería, ropa urbana y zapatillas en Concepción, Chile.

## Características

- Reserva de horas online
- Gestión de servicios de barbería
- Catálogo de productos urbanos
- Interfaz moderna y responsiva

## Tecnologías

- React con TypeScript
- Vite
- Tailwind CSS
- React Query para gestión de estado
